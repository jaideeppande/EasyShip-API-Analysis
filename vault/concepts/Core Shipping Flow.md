# Core Shipping Flow

The happy path for shipping with Easyship:

1. Validate addresses → [[Addresses]]
2. Request rates → [[Rates]]
3. Create shipment (+ buy label) → [[Shipments]]
4. Pick courier / services → [[Couriers]] · [[Courier Services]]
5. Schedule pickup → [[Pickups]]
6. Generate manifest → [[Manifests]]
7. Track delivery → [[Trackings]]
8. Get notified → [[Webhooks]]
